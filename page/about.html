About

My name is Izaak Kuipers, a hobbyist developer since I was young. I have been interested
in this topic since the age of 6, and have grown to know a lot about it since.


Over the years I have learned a lot of skills, of which my biggest one is web development, using HTML, CSS and JS.
Since the beginning of 2021 I learned about Electron, and I have created a lot of great projects in it since,
with more to come in the future. My main Electron project is ArcOS, a Operating System Environment that runs
on top of Windows or Linux, like how Windows used to run on top of MS-DOS.


But web development is not my only skill, I also know a lot about the Windows operating system, of which the
greater part was self-taught. I use this knowledge to help my friends and family, and other people that request
my help. I know e.g. what system files are required to run Windows, how the images are
stored inside the DLL files, and more.


So, if you want to know what I have managed to create over time, head on over to my projects page, which will list
mainly the bigger projects nicely on display for you to enjoy. Have fun experimenting for yourself, and let me know
what you think about this website!
